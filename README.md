### 主题特色
- 采用经典的双链样式，[[双向连接]]，带给大家熟悉的操作体验
- 设计参考QQ浏览器，包括标签页和面包屑
- 超级块之间的间隔拉大，更加美观
- 自定义属性和 zhang-light 主题一致
- 隐藏思源笔记的顶栏按钮，鼠标悬浮可以显示（代码来自于Roy制作的notion-theme主题）
- 加入集市中"abbj"设计的搜索栏左右分栏的代码，搜索起来更方便
- 网址链接前面加入网站图标，并且链接文字颜色与网站图标颜色一致
- 配色非常的好看
- 属于本主题的独特设计：
  - 给引用添加外观中的4种样式颜色，即可出现专属便利贴
